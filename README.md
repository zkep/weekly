# weekly
anyfun weekly  技趣周刊


#### Why? 为什么开始写周刊呢？
    原因很多，比如
    1. 技术或生活备忘录
    2. 锻炼自己的写作能力
    3. 从业了许多年，也有一些想输出的冲动了（持续学习）
    4. 养成一个总结的习惯
    后面想到其他的再补充

#### What to do？ 做什么？
    目前也不确定做什么，随缘或跳票的可能性比较大
    大概的方向分享技术上有趣的东西，或者也夹带私货
    

#### How do？ 怎么做？
    每周一发布
    学会一个小技巧
    相信时间的力量，做长期主义者
